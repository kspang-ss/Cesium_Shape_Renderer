# Cesium Shape Renderer

A CesiumJS application for rendering and toggling geometric shapes on a 3D globe.

**Live Demo:** https://kspang-ss.github.io/Cesium_Shape_Renderer/

## Features

- Renders shapes (rectangles, spheres, squares, triangles) on a Cesium 3D globe
- Toolbar with checkboxes to toggle shape visibility

## Tech Stack

- [CesiumJS](https://cesium.com/cesiumjs/) - 3D geospatial visualization
